# NusaResto

NusaResto adalah sebuah aplikasi restoran fiksi yang dibuat sebagai projek
belajar Full-stack.

(Projek ini masih WIP)

## Pengembangan

Untuk backend, masuk ke folder backend dan jalankan `go run main.go`. Service
backend akan berjalan di port `3000`.

Untuk frontend, masuk ke folder frontend dan jalankan `pnpm run dev`. Aplikasi
web akan berjalan di port `4321`.
